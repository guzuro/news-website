# News website

## App features

 - Add articles / categories and it metadata (description, tags..) from Strapi admin;
 - Watch all articles or by categorie;

## Technologies
**Frontend:**
 - [x] Astro.build v4 JavaScript
 - [x] Typescript
 - [x] Tailwind

**Backend:**
 - [x] Strapi as HCMS

 ## Run application

Strapi (creare admin user and add some content then)
```bash
  yarn develop
```

Client
```bash
  yarn dev
```
